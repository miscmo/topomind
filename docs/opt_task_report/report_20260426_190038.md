## 本轮新增发现

### 🔍 理论观察（暂不修复）

1. **ContextMenu.tsx Hooks 规则顺序**
   - `useMemo` 和 `useState` 在 `if (!visible) return null` 之前已有正确顺序
   - 但所有 hooks 在条件 return 之前，违反了 React "only call hooks at top level" 规则中的"不要在条件语句中调用 hooks"条款
   - 功能上无 bug，但理论上可能影响 React 未来版本或 React DevTools
   - **建议**：重构为 `if (!visible) { return null }` → `useMemo(...)` → `useState(...)` 顺序
