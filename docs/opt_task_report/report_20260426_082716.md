## 【三、详细修复说明】

### 3.1 SearchBar.tsx — 添加 Enter 键处理

**问题**：`onChange` 处理输入变化，但回车键无任何响应，用户无法通过按 Enter 执行搜索（实际搜索由外部 `onSearchChange` 回调触发，此处记录日志）。

**解决方案**：
```typescript
// 新增 onKeyDown handler
const handleKeyDown = (e: React.KeyboardEvent<HTMLInputElement>) => {
  if (e.key === 'Enter') {
    logAction('搜索:回车', 'SearchBar', { query: searchQuery })
  }
}

// input 元素添加
<input ... onKeyDown={handleKeyDown} />
```

### 3.2 Breadcrumb.module.css — 路径溢出截断

**问题**：容器有 `white-space: nowrap` 但无 `max-width`，路径过深时横向溢出覆盖其他 UI 元素。

**解决方案**：
```css
/* 容器约束最大宽度 */
.breadcrumb {
  max-width: calc(100vw - 240px);
  overflow: hidden;
  white-space: nowrap;
}

/* 链接和当前项单独截断 */
.link, .current {
  max-width: 120px;
  overflow: hidden;
  text-overflow: ellipsis;
}
```
