## 四、功能优化详情

### B-D1（CRITICAL）— useResizePanel 面板拖拽完全失效 ✅

**问题描述**：`handleMouseMove` 是 `useCallback` 包裹的回调，内部通过 `isResizing` state 判断是否继续执行。但 `useCallback` 在创建时捕获的是 `isResizing` 的**值快照**，而非实时引用。

**原因分析**：React 的 `useState` 更新触发组件重渲染，生成新的 `handleMouseMove` 回调。但 mousemove 事件触发时调用的仍是旧的回调引用——因为事件监听器在 useEffect 中注册时捕获了当时的回调版本。

**解决方案**：使用 `useRef(false)` 替代 `useState(false)`，`isResizingRef.current` 读写，ref 的 `.current` 属性在组件整个生命周期中保持同一引用。

**验证方式**：
1. 打开 TopoMind，进入任意知识库图谱页面
2. 找到右侧面板的 resize 拖拽手柄（分隔线）
3. 按住鼠标左键拖拽，面板宽度应实时变化
4. 拖拽过程中鼠标变为 `col-resize` 光标
5. 释放鼠标，拖拽状态结束

### F1（工具栏缺少适应视图按钮）✅

**问题描述**：工具栏仅有放大/缩小按钮，用户无法快速恢复默认视图。

**解决方案**：在 `Toolbar.tsx` 中添加 fitView 按钮（⊙），调用 `fitView({ padding: 0.2, duration: 300 })`。

### F2（右键菜单缺少 Escape 键关闭）✅

**问题描述**：ContextMenu 组件不支持 Escape 键关闭，与 ConfirmModal、PromptModal 行为不一致。

**解决方案**：在 ContextMenu 中添加 useEffect，监听 `keydown` 事件，当按键为 `Escape` 时调用 `hideCM()`。

### F3/F4（被注释的导入未清理）✅

**问题描述**：GraphPage.tsx 中 SearchBar 和 GitPanel 的 JSX 被注释，但导入语句未同步移除。

**解决方案**：移除 `SearchBar` 和 `GitPanel` 的未使用导入。
